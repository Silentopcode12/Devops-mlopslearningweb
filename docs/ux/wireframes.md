# Key Screen Wireframes

## 1. Role Dashboard
- Top: company switcher, search, notifications.
- Left nav: Employees, Attendance, Leaves, Payroll, Performance, ATS, Expenses, Helpdesk.
- Main widgets: headcount, pending approvals, payroll status, attendance anomalies.

## 2. Employee Management
- Table with filters (department, role, status).
- Side panel for profile + documents + organization node.
- Actions: create, edit, deactivate, export.

## 3. Attendance & Shift
- Daily timeline and map pins for remote check-ins.
- Shift assignment + geofence validation status.
- Missing check-out queue for managers.

## 4. Leave Approval Desk
- Policy-aware leave request cards.
- Calendar overlay + overlap alerts.
- One-click approve/reject with audit log entry.

## 5. Payroll Run
- Stepper: draft -> validate -> lock -> publish.
- Tax/deduction summary panel.
- Payslip generation and CSV/PDF export.
